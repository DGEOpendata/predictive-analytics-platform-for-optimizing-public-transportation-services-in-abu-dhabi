# Predictive Analytics Platform for Optimizing Public Transportation Services in Abu Dhabi

## Overview
This project is a Python-based predictive analytics platform designed to optimize public transportation services in Abu Dhabi. By leveraging the `Public Transportation User Data - 2026` dataset, the platform provides actionable insights into transportation usage patterns and predicts future trends.

## Features
- **Data Cleaning and Preparation**: Automatically preprocess raw dataset.
- **Visualization**: Generate graphs and charts to illustrate public transportation trends.
- **Predictive Analytics**: Use machine learning models to forecast future passenger counts.
- **Export**: Save cleaned data for further analysis.

## Requirements
To run this project, you will need:
- Python 3.8+
- Required Python libraries:
  - pandas
  - numpy
  - matplotlib
  - statsmodels

You can install the required Python libraries using the following command:
bash
pip install pandas numpy matplotlib statsmodels


## Dataset
Download the dataset from [Public_Transportation_User_Data_2026.csv](http://opendata.abudhabi/public_transportation_user_data_2026).

## Usage
1. Clone the repository:
bash
git clone https://github.com/yourusername/predictive-transportation-platform.git

2. Navigate to the project directory:
bash
cd predictive-transportation-platform

3. Place the dataset file in the project directory.
4. Run the script:
bash
python transport_analytics.py


## Output
- Visualizations of historical passenger trends.
- Forecasts for passenger counts for the next 12 months.
- Cleaned dataset saved as `Cleaned_Public_Transportation_User_Data_2026.csv`.
